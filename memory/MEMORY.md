# 工作记忆系统

## 每日记录规则

每天工作结束前，自动创建/更新 `memory/YYYY-MM-DD.md`，记录：
1. 当天完成的主要工作
2. 关键决策和结论
3. 待办事项（明天继续）
4. 项目状态更新

## 每周整理

每周五整理本周记忆，将重要信息归档到 `MEMORY.md`：
- 长期项目进展
- 用户偏好和习惯
- 重要联系人/账号信息
- 学到的经验教训

## 记忆文件结构

```
memory/
├── 2026-04-13.md    # 今天的工作记录
├── 2026-04-12.md    # 昨天
├── 2026-04-11.md    # 前天
└── ...
```
